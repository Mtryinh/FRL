Settings
⚡ Activity Types
Customise the activity types available when logging an interaction with a contact or venue. The defaults (Call, Meeting, Note) cannot be removed.
◬ Venue Info Fields
Define custom fields that will appear on every venue's profile under an "Info" tab. Choose a field type and label — you can then fill in the values per venue.
⚡ Auto Follow-Up Triggers
When a matching activity type is logged against a contact, a follow-up task is automatically created on the calendar. Set a default delay and task label per trigger.
👤 Team Members
Add the names of people who can take bookings. These appear in the "Taken By" dropdown on every booking form.
◎ Contact Types
Manage the contact types available when adding or editing a contact. All types are fully editable — you can add, rename or remove any of them.
🎉 Event Types
Manage the event types available when logging a booking. These appear in the Event Type dropdown on every booking form.
📡 Lead Sources
Manage the lead source options available when adding or editing a contact. Keeping these as a managed list ensures clean, consistent data.
🥶 Cold Tag Settings
Configure how cold tags are applied to contacts based on days since last activity. Tags are applied automatically on app load when enabled.
Enable cold tagging
When off, no cold tags are added or updated on load
Tier 1 — Warm to Cold
Tier 2 — Cold to Very Cold
✦ New Contact Defaults
Set default values that pre-populate when adding a new contact. You can always override these per contact.
🌍 Cities & Venues
Manage the cities and venues available across your multi-city CRM. Cities are used to scope and filter all data. Venues belong to cities.
Cities
Venues
Venues are managed in the Venues tab. Each venue has a city field — assign a city there to link it.
Area Types
Manage the types available when adding or editing an area within a city. Multiple types can be assigned to each area.
Event Calendar — Event Types
Manage the event types available when adding events to the Event Calendar.
✉ Email Templates
Create reusable email templates for common outreach. Templates can be loaded into the booking confirmation email generator.
